To maximize the learning output, it is strongly advised to do the two exercises beforehand!

To test your understanding of the various subjects, you're required to do Problem 1: Use linear/quadratic functions for verification and Exercise 21: Simulate an elastic pendulum. These comprise the first compulsory exercise of this semester. You should submit your Python implementations in a directory under your root GitHub repository. See the webpage for instructions. The submission deadline is noon Monday September 7. Yapi will lead the assessment event at 14:15-16:00 on the same day, in room Perl.

For the maximum learning output, you're strongly encouraged to solve these two exercises beforehand.
