How to submit an exercise or project

Hans Petter Langtangen (hpl at ifi.uio.no)

Exercises or projects are not accepted as email attachments (because that is a too inefficient procedure with storing and archiving files). Instead, all files associated with exercises or projects must exist in some Git repository on GitHub. This is the way people work with software and technical documents in 2014!

  • Go to github.com and create a user
  • Fill out this form about your GitHub account
  • In github, click on settings in the top right corner
  • Go to emails
  • Add your university email and verify it.
  • Go to SSH keys and open the manual for creating a SSH key. Follow the manual.
  • Go to education.github.com, click on Request a discount and fill out the form.
The course will automatically create a repository for you with a name INF5620-yourname. You are supposed to put all your exercises and projects in this repository. If you have any questions about this, please send an email to inf5620@gmail.com.
